# Spring Boot Template
This repository contains a starting RESTful API project template using Java Spring Boot. You can fork or clone this repository and start developing your API.

Project template contains a User entity and register, login operations are already implemented. You can change User entity based on your requirements.

Project template uses PostgreSQL as database, if you want to use another database, you should change dependencies in pom.xml and application.properties based on your database connection requirements.

Endpoints are JWT Authenticated so you don't need to do any configurations to authenticate your endpoints but feel free to do changes based on your requirements of API.

# IMPORTANT NOTE

Before starting development, please create a secret.properties file in resources folder to store your database connection informations and other secret informations (token, url links etc.)
